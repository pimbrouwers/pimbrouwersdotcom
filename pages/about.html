About Me

I'm Pim, a software engineer living in Toronto, Canada. I focus on data systems in the F# programming language and building relational databases.

I prefer to write my own code. I know that sounds silly, but I don't like prebuilt things. I believe:

  • in starting small and finishing big;
  • that you should understand your data first;
  • in simple solutions;
  • that reflection should be avoided;
  • in immutability;
  • in composition;

When I'm not working at the NHLPA, I contribute to open-source, ramble on Twitter, write occasionally, try to help other developers write better code and dabble in photography.
